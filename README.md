# FFmpeg-GUI
A GUI video processing tool based on FFmpeg

FFmpeg GUI combines the convenience and clarity of GUI interface with FFmpeg's great capability. This program is developed using Python and PyQt5 GUI framework.

## Dependency & Requirements

### Dependency

* FFmpeg 4.2.2 or later
* PyQt5
* Python 3.4 or later

**Notice: These dependencies are only needed when you run program directly from `.py` file**

### System Requirements
* Apple macOS 10.14 or later
* Microsoft Windows 7 or later

**Notice: Linux systems hasn't been tested for compability**
